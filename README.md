Easy Thai Trip 🇹🇭✈️

Easy Thai Trip เป็นแอปพลิเคชันสำหรับช่วยวางแผนการท่องเที่ยวในประเทศไทย โดยผู้ใช้สามารถเลือกจังหวัด ค้นหาสถานที่ท่องเที่ยว และจัดแผนการเดินทางได้อย่างสะดวกผ่านแอปพลิเคชันบนมือถือ

ระบบถูกพัฒนาขึ้นเพื่อช่วยลดเวลาในการค้นหาข้อมูลจากหลายแหล่ง และช่วยให้ผู้ใช้สามารถจัดการแผนการเดินทางได้ภายในแอปเดียว

✨ Features
🗺️ ค้นหาสถานที่ท่องเที่ยว
ผู้ใช้สามารถค้นหาสถานที่ท่องเที่ยวจากจังหวัดต่าง ๆ ของประเทศไทยได้อย่างสะดวก

📍 เลือกสถานที่ตามหมวดหมู่
ระบบมีการจัดหมวดหมู่สถานที่เพื่อช่วยให้ผู้ใช้ค้นหาสถานที่ได้ง่ายขึ้น เช่น
* ธรรมชาติ
* วัด
* จุดชมวิว
* ทะเล
* คาเฟ่
* สถานที่ยอดนิยม

🎡 วงล้อสุ่มจังหวัด
ฟีเจอร์สุ่มจังหวัดสำหรับผู้ใช้ที่ยังไม่รู้ว่าจะไปเที่ยวที่ไหน โดยระบบจะสุ่มจังหวัดเพื่อช่วยเป็นไอเดียในการท่องเที่ยว

📅 สร้างแผนการเดินทาง (Trip Planner)
ผู้ใช้สามารถ
* เพิ่มสถานที่ลงในแผนการเดินทาง
* จัดลำดับสถานที่
* แบ่งแผนการเดินทางตามวัน

🔎 ระบบค้นหาสถานที่
สามารถค้นหาสถานที่ได้จาก
* ชื่อจังหวัด
* ชื่อสถานที่

🖥️ Technology Stack
Frontend
เทคโนโลยีที่ใช้พัฒนาแอปพลิเคชัน
* React Native – ใช้สำหรับพัฒนา Mobile Application
* Expo – เครื่องมือช่วยพัฒนาและทดสอบแอป
* TypeScript – ภาษาโปรแกรมที่ช่วยให้โค้ดมีโครงสร้างที่ชัดเจนและลดข้อผิดพลาด

UI / Design
เครื่องมือที่ใช้ในการออกแบบหน้าตาแอป
* React Native Components
* StyleSheet สำหรับจัด Layout และกำหนด Style
* Ionicons สำหรับแสดงไอคอนในแอป
* Safe Area Context สำหรับจัด Layout ให้เหมาะสมกับหน้าจอมือถือ

Data
ข้อมูลสถานที่ท่องเที่ยวถูกจัดเก็บในรูปแบบ
JSON Dataset
ซึ่งเก็บข้อมูลเกี่ยวกับสถานที่ เช่น
* จังหวัด
* ชื่อสถานที่
* หมวดหมู่
* รายละเอียดสถานที่

Tools
เครื่องมือที่ใช้ในการพัฒนา
* Visual Studio Code
* GitHub (Version Control)
* Node.js
* Expo CLI
  
🏗️ System Architecture
แอปพลิเคชัน **Easy Thai Trip** ถูกพัฒนาในรูปแบบ **Mobile Application** โดยใช้ React Native และ Expo ซึ่งช่วยให้สามารถพัฒนาแอปข้ามแพลตฟอร์มได้
โครงสร้างของระบบแบ่งออกเป็น 3 ส่วนหลัก
1️⃣ User Interface (Frontend)
2️⃣ Data Layer (Data Management)
3️⃣ Application Logic

1️⃣ User Interface (Frontend)
ส่วนติดต่อผู้ใช้ถูกพัฒนาด้วย
* React Native
* Expo Framework
* TypeScript
React Native เป็น Framework สำหรับพัฒนา Mobile Application ด้วย JavaScript หรือ TypeScript ซึ่งสามารถสร้างแอปที่ทำงานได้ทั้ง Android และ iOS

Expo เป็นเครื่องมือที่ช่วยให้การพัฒนา React Native ง่ายขึ้น เช่น
* การรันแอปแบบรวดเร็ว
* การจัดการ dependencies
* การทดสอบแอปผ่านมือถือ

ภายใน UI มีการใช้ React Components เพื่อแบ่งโครงสร้างหน้าจอ เช่น
* หน้า Home (หน้าหลัก)
* หน้าเลือกสถานที่
* หน้าแผนการเดินทาง
* หน้ารายละเอียดสถานที่

# 2️⃣ Data Layer (Data Management)
ข้อมูลสถานที่ท่องเที่ยวถูกจัดเก็บในรูปแบบ JSON Dataset
ไฟล์ข้อมูลจะอยู่ในโฟลเดอร์
data/places.ts

ภายในไฟล์จะเก็บข้อมูลสถานที่ เช่น
* id
* name
* province
* category
* description

ตัวอย่างโครงสร้างข้อมูล
ts
{
  id: 1,
  name: "ดอยอินทนนท์",
  province: "เชียงใหม่",
  category: "ธรรมชาติ",
  description: "จุดชมวิวที่สูงที่สุดในประเทศไทย"
}

ข้อดีของการใช้ JSON
* โครงสร้างเข้าใจง่าย
* โหลดข้อมูลได้รวดเร็ว
* เหมาะกับโปรเจกต์ขนาดเล็ก

3️⃣ Application Logic
ส่วนนี้เป็นตรรกะการทำงานของระบบ เช่น
* การค้นหาสถานที่
* การสุ่มจังหวัด
* การเพิ่มสถานที่ลงในแผนการเดินทาง
* การจัดลำดับสถานที่ในแต่ละวัน

การจัดการข้อมูลภายในแอปใช้ **React Hooks** เช่น
* `useState`
* `useEffect`

เพื่อจัดการสถานะของข้อมูล เช่น
* เก็บสถานที่ที่ผู้ใช้เลือก
* เก็บแผนการเดินทางของผู้ใช้
